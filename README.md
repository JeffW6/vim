# Vim Config

## vimrc config.

### Installation

- Download directly.
```
# Download vimrc.
wget https://raw.githubusercontent.com/JeffW6/vim/master/vimrc

# if ~/.vimrc file isn't exist.
mv vimrc ~/.vimrc
```

- Alternatively clone the repo.
```
cd ~/.vim
git clone git clone https://github.com/jeffw6/vim.git

# Edit ~/.vimrc, and add this:
so ~/.vim/vim/vimrc
```
